Mini Workshops

Open Source & Making your first OS contribution & GSOC

"Dive into the world of open-source software and transform from a user to a contributor.

In this interactive workshop, you'll navigate the ecosystem of open-source projects and learn how to make your mark.

Covering the basics of choosing a project, understanding issues, and making your first pull request, this session is your stepping stone into the grand arena of open-source contributions.

Plus, tips on how to get selected for the prestigious Google Summer of Code (GSOC)!"

Git & All of Git

"Unlock the full potential of version control with Git in this hands-on session.

We'll guide you through branching strategies, conflict resolution, and repository management, ensuring you emerge as a Git guru.

Collaborate with confidence and take control of your development workflow, one commit at a time!"

Get Started with React JS

"React JS is reshaping the way we build the web.

In this immersive workshop, jumpstart your journey with React by building dynamic user interfaces that come alive.

We'll cover JSX, state management, and the component lifecycle, giving you the practical skills to create reactive and efficient web apps."

Get Started with React Native

"Bring your mobile app ideas to life with React Native!

This hands-on workshop will have you creating cross-platform mobile applications that feel truly native.

Learn about React Native components, navigation, and state management in an interactive environment that will leave you with a functional app on your device!""

Build Your Own URL Shortener

"Ever wondered how to create a service like Bit.ly?

Discover how in our practical workshop where you will build your very own URL shortener from scratch.

Get to grips with backend development, RESTful APIs, and database integration, as we take you through a project that encapsulates the magic of full-stack development.""

Topics of CSS to Build Any Website

"Unravel the secrets of web design with a CSS deep dive that will empower you to build visually stunning websites.

Tackle topics like Flexbox, Grid, custom properties, and advanced selectors.

Expect to leave with the confidence to turn any design mockup into a fully functional and responsive website layout.""

APIs Development and Deployment

"APIs are the building blocks of the modern web.

This workshop offers a comprehensive look into creating and deploying robust APIs.

Learn about REST principles, authentication, rate limiting, and documentation.

We’ll also cover tools and practices for testing and deploying your APIs, so you can connect the world with your services."

Deploy Your Own ML Model

"Machine learning isn't just for data scientists.

In this session, we'll demystify the process of creating and deploying a machine learning model.

Covering data preprocessing, model selection, and deployment using platforms like TensorFlow and PyTorch, you’ll get a real taste of AI development that’s directly applicable in the industry."

Responsive Design & Hosting Your Website

"Responsive design is no longer optional.

In this interactive session, we'll teach you the principles of making your website look great on any device.

We’ll also walk you through hosting options, domain selection, and deployment strategies, so your sites are not just live but lightning fast and secure."

Prompt Engineering

"As AI language models become ubiquitous, crafting the right prompts is an art form.

Learn the intricacies of prompt engineering to effectively communicate with AI and extract the desired output.

This session will cover techniques for designing prompts that result in creative, accurate, and useful AI responses."

The Beauty of JS

"JavaScript is the heart of interactive web experiences.

This workshop celebrates the versatility of JS, covering everything from fundamental concepts to advanced features like async/await, closures, and the event loop.

You’ll write code that breathes life into web pages, making them interactive and fun!"
